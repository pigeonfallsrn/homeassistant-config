## MUSIC — Music Assistant
- media_player.kitchen_2 = MA entity (music_assistant platform) wrapping Sonos Beam
- MA play_media: bare playlist ID only e.g. 37i9dQZF1DWZd79rJ6a7lp
- NOT: spotify://playlist/ID → 500 error
- radio_mode:true = infinite genre radio from seed playlist
- Plain text e.g. media_id:"chill" does fuzzy library search

## FK — A9+ Stability Settings (CRITICAL)
- Keep Screen On → ON (prevents rebuild-on-wake on permanently mounted tablet)
- Keep in Memory / Don't Pause Web App → ON (prevents Samsung WebView kill)
- Auto Reload after Page Error → 30 seconds
- Samsung Developer Options: Don't keep activities → OFF (default ON = kills WebView)
- Motion detection crashes FK on A9+ → use Apollo mmWave + HA automation for screen wake instead
