
## Claude MCP Integration (NEW 2026-02-16)

**Quick Start:** `hac mcp` opens Desktop with live entity access

**Interfaces:**
- Desktop: Local MCP (primary) - real-time control
- Code: MCP + file access - automation editing
- Web: No MCP (OAuth pending)

**Usage Pattern:**
- MCP: Current states, control, testing
- Files: Editing, config, git
- Workflow: Query(MCP) → Edit(file) → Test(MCP) → Commit
